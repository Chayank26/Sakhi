# Sakhi – A Digital Support Platform for Women

## Vision Document

### 1. Project Overview
Sakhi is a web-based platform designed to provide women with a safe, supportive, and accessible digital space. The platform centralizes access to community support, job opportunities, and safety features, addressing the lack of trusted and unified digital resources for women.

### 2. Problem Statement
Women often face fragmented access to essential resources such as employment opportunities, government schemes, and safe community support. Existing platforms are either unsafe, unverified, or scattered across multiple sources, making it difficult to seek help quickly and securely.

### 3. Target Users (Personas)

**Persona 1: College Student**
- Age: 18–23
- Goals: Find internships/jobs, seek guidance
- Pain Points: Unsafe platforms, lack of verified information

**Persona 2: Working Professional**
- Age: 23–35
- Goals: Community interaction, career growth
- Pain Points: Limited trusted networks

**Persona 3: Homemaker**
- Age: 25–45
- Goals: Access government schemes and support
- Pain Points: Lack of awareness and digital guidance

### 4. Vision Statement
To build a secure and inclusive digital ecosystem that empowers women by providing trusted access to opportunities, community support, and safety resources.

### 5. Key Features / Goals
- Secure user authentication  
- Community discussion forum  
- Job opportunity listings  
- Emergency SOS feature  
- Clean and accessible user interface  

### 6. Success Metrics
- Number of registered users  
- Active community participation  
- User retention rate  
- Successful SOS feature activations (simulated)  

### 7. Assumptions & Constraints

**Assumptions**
- Users have internet access  
- Users are comfortable with basic web navigation  

**Constraints**
- MVP-level feature implementation  
- No real emergency service integration
